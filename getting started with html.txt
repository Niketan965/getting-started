Q4
tags and elements are the fundamental concepts used to define the content and structure of a webpage.
In HTML, "tags" and "elements" are key concepts for structuring web page content. HTML tags, enclosed in angle brackets, 
define the structure and elements of a webpage. They are case-insensitive and come in pairs, with an opening tag and, if applicable, a closing tag. Elements consist of tags, content, and optional attributes and represent various parts of a webpage, such as headings, paragraphs, links, and images. HTML tags and elements play a crucial role in creating the structure and appearance of web content.

the doctype decleration is a special structure in the starting of html that defines the version of Html and to tell the web browsers a set of rules while rendering content on it.
The doctype declaration is not an HTML tag and doesn't affect the appearance of the web page; rather, it influences how the document is processed.
Including a correct and specific doctype declaration is important because:

Compatibility, Validation, Document Mode, Future Compatibility
